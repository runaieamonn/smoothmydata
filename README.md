# Smooth My Data

This is the implementation code for a static website that showcases the [Smoothish][1] JavaScript library.

It allows users to paste or upload time-series data and smooth it using any of the options that the Smoothish library provides.

Both the unsmoothed input and smoothed output are graphed.

The smoothed data can be downloaded.

[1]: https://github.com/eobrain/smoothish
